            <li>Créez un environnement virtuel:</li>
                <ul><li>python -m venv env</li></ul>
            <li>Puis activation de l'environnement virtuel:</li>
                <ul><li>Sur Windows:</li></ul>
                env\Scripts\activate

                source/env/Scripts/activate
